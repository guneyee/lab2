# Project Brief: Personal Task Board

## Summary
Build a frontend-only personal task board for a solo developer to manage work across 2-3 concurrent projects.

## Problem
Developers need a lightweight task board for daily planning and execution without adopting heavy project management tools like Jira.

## Primary User
- Solo developer managing personal and project tasks
- Typical workload spans 2-3 active projects at any given time

## Core Features
- Kanban board with 3 columns: To Do, In Progress, Done
- Drag and drop task movement between columns
- Keyboard shortcuts for fast task operations

## Technology Constraints
- React + Vite
- No backend
- All data stored in browser localStorage

## Product Constraints
- Must run entirely in the browser
- Must preserve tasks between browser refreshes
- Must remain simple and fast for daily individual use
